# Stockhouse Concert Agent Guide

## Description

This is a website for the Janis Stockhouse tribute concert.

## Commands

- **Install dependencies**: `pnpm install`
- Format code: `pnpm format`
- Live dev server: `pnpm serve`
- Build site: `pnpm build`
- Generate program PDF: `pnpm pdf`

## Architecture

- EleventyJS site managed with pnpm

## Site Design

- Primary accent color: #800000
- Secondary accent color: #ffd700
- Design philosophy: Clean, modern, and responsive
- CSS Libraries: Tailwind CSS for utility-first styling
